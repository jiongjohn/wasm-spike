## 申请步骤(2026-06-29 核于官方文档；以平台实际流程为准)

**官方链接:**
- 主入口(注册/登录控制台):https://developer.open-douyin.com/
- 注册与入驻:https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/guide/minigame/register
- 创建与信息完善:https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/guide/minigame/improve-information
- 小游戏文档总览:https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/guide/overview

**步骤:**
1. 访问 **developer.open-douyin.com**,用**手机号 / 邮箱 / 抖音号**注册并登录。
2. 完成**主体认证(实名/资质)** —— 个人 or 企业(个人能否接激励广告见上方待核实 Q1)。
3. **创建小游戏**:开发引擎选 **Godot**、填小游戏名称/联系人/手机/邮箱 → 勾协议 → 创建。
4. 🔑 **取 AppID**:**开发管理 → 开发设置** 查看 AppID → **填回 spike 导出预设的 App ID 字段**(替换 dummy `ttspike0000000000`)→ 重新导出 → 拖进 Douyin IDE 应能越过「正在初始化」、渲染出 spike(验 ADR-0007 P5/P3-6/QQ-ADR8-01)。
5. 申请小游戏类目;记录类目对变现方式的限制。
6. 在「变现 / 流量主」模块申请激励视频广告权限;记录资质要求与分成(#17)。
7. Godot 开发者插件(ttsdk 1.0.3)与 Douyin 开发者工具均已就位(见本会话 spike 记录)——本步已完成。

> **当前关键路径**:第 1–4 步(注册→认证→建小游戏→取 AppID)是 spike 收尾(ADR-0007 转 Accepted)的**唯一阻塞项**。第 5–6 步(类目/广告资质)可并行,服务 VS 阶段 #14/#17。
